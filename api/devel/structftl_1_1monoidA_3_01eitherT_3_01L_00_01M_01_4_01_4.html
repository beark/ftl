eitherT< L, M >  e2 
)
static

Evaluate two alternatives.

If e1 wraps a right value, it is instantly returned. Otherwise, e2 is checked for rightness. If both e1 and e2 wrap left values, they are combined using monoid<L>::append and a new left value (embedded as if by monad<M>::pure) is returned.
