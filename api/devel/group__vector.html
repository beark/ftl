Detailed Description

Implementations of concepts, utility functions, and more for std::vector.

#include <ftl/vector.h>

This module adds the following concept instances to std::vector:

  • Monoid
  • Foldable
  • Functor
  • Applicative Functor
  • Monad
  • Zippable
